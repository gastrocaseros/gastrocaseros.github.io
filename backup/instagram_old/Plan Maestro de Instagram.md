## 8. Sistema de Automatización con Make

### 8.1 Arquitectura del Sistema (Make.com, plan gratuito)

1. **Google Sheets (Panel de Contenidos):**
   - Columnas: FechaProgramada, Estado, Caption, Hashtags, ImagenDriveID.
2. **Escenario 1 (Revisión - Lunes 9:00):**
   - Lee Google Sheets para ver posts listos. Manda resumen al Gmail.
3. **Escenario 2 (Publicación - Mar/Mié 17:50):**
   - Descarga imagen de Drive → Postea en Meta Graph API → Marca como "Publicado" en Sheets.
4. **Escenario 3 (Story Turnos - Mié 18:00):**
   - Sube Story pre-armada "Turnos Jueves disponibles".

### 8.2 Prerequisitos Meta Graph API
1. Facebook Page vinculada al IG.
2. developers.facebook.com → App Business → Instagram Graph API.
3. Generar token de larga duración con permisos: `instagram_basic, instagram_content_publish, pages_read_engagement`.
4. *Renovar cada 55 días (alarma en Google Calendar).*

> **Alternativa sin Make:** Programar vía **Meta Business Suite** (nativa de Instagram) para el primer mes.
